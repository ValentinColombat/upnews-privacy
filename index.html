Politique de Confidentialité - UpNews

Dernière mise à jour : 11 février 2026

Introduction

UpNews notre application respecte votre vie privée. Cette politique de confidentialité explique quelles données nous collectons, comment nous les utilisons, et vos droits concernant vos données personnelles.

1. Données Collectées

1.1 Données que vous nous fournissez

Lorsque vous créez un compte, nous collectons :

  • Votre adresse e-mail
  • Votre nom (prénom et nom)
  • Votre photo de profil (si connexion via Google/Apple)
  • Votre identifiant unique (Google User ID, Apple User ID, ou compte UpNews)

Lorsque vous utilisez l'application, nous collectons :

  • Votre compagnon sélectionné
  • Vos points d'expérience (XP)
  • L'historique des news que vous avez écoutées
  • Vos préférences de notifications

1.2 Données collectées automatiquement

Nous collectons automatiquement certaines informations techniques :

  • Adresse IP
  • Type et version de votre appareil iOS
  • Date et heure de vos connexions
  • Données d'utilisation de l'application

2. Comment Nous Utilisons Vos Données

Nous utilisons vos données pour :

  • Fournir le service : Authentification, sauvegarde de votre progression, personnalisation
  • Améliorer l'application : Comprendre l'utilisation pour améliorer les fonctionnalités
  • Communications : Notifications quotidiennes (si activées)
  • Sécurité : Prévenir la fraude et assurer la sécurité

3. Partage de Données avec des Tiers

Nous ne vendons jamais vos données personnelles. Nous partageons vos données uniquement avec :

3.1 Google (Authentification)

Si vous utilisez Google OAuth, Google collecte et traite vos données selon sa propre politique : https://policies.google.com/privacy

3.2 Apple (Authentification)

Si vous utilisez Apple Sign In, Apple traite vos données selon sa politique : https://www.apple.com/legal/privacy/

3.3 Supabase (Hébergement)

Vos données sont stockées sur des serveurs Supabase (conformes RGPD) : https://supabase.com/privacy

4. Conservation des Données

Nous conservons vos données aussi longtemps que vous avez un compte actif. Si vous supprimez votre compte, toutes vos données sont définitivement supprimées sous 30 jours.

5. Vos Droits (RGPD)

Vous disposez des droits suivants :

  • Droit d'accès : Demander une copie de vos données
  • Droit de rectification : Corriger des données incorrectes
  • Droit à l'effacement : Supprimer votre compte et données (via Paramètres)
  • Droit d'opposition : Vous opposer au traitement
  • Droit à la portabilité : Recevoir vos données dans un format structuré

6. Suppression de Compte

Vous pouvez supprimer votre compte à tout moment :

  1. Ouvrez UpNews
  2. Allez dans "Profil"
  3. Appuyez sur "Supprimer mon compte"
  4. Confirmez la suppression

Toutes vos données seront définitivement supprimées sous 30 jours.

7. Sécurité

Nous prenons la sécurité très au sérieux :

  • Connexion HTTPS chiffrée
  • Données stockées sur serveurs sécurisés
  • Authentification sécurisée (Google OAuth, Apple Sign In)
  • Accès limité aux données

8. Protection des Mineurs

UpNews est destiné aux utilisateurs de 4 ans et plus.

9. Modifications

Nous pouvons mettre à jour cette politique. Vous serez informé des changements importants via l'application ou par e-mail.
